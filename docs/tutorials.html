基础教程

从零开始掌握 Python 的核心概念。

变量与数据类型

在 Python 中，你可以使用变量保存数字、字符串、列表、字典等数据类型。

x = 10
name = "Alice"
numbers = [1, 2, 3]

控制流

使用条件语句和循环可以控制程序执行的流程。

for i in range(5):
    if i % 2 == 0:
        print(i, "是偶数")
    else:
        print(i, "是奇数")

函数与模块

函数可以封装重复使用的逻辑，模块可以组织代码并在多个文件之间共享。

def greet(name):
    return "你好, " + name

import math
result = math.sqrt(16)
print(greet("Bob"))
print("16 的平方根是", result)
