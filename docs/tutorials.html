Python Tutorials

Learn the fundamentals of Python through concise tutorials.

Variables and Data Types

Python variables are dynamically typed. You can assign numbers, strings, lists, dictionaries, and more:

x = 10
name = "Alice"
numbers = [1, 2, 3]

Control Flow

Use if, elif, and else to control the flow of your program:

age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

Functions

Encapsulate logic into reusable functions using the def keyword:

def greet(name):
    return f"Hello, {name}!"

print(greet("World"))

Modules

Python modules allow you to organize code into separate files. Import modules using the import statement:

import math
print(math.sqrt(16))
